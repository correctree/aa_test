# 人物認識 背景スライドショー

## 使い方

1. `index.html` をGitHub PagesなどのHTTPS環境にアップロードします。
2. iPhone / iPadではSafariでURLを開きます。
3. 「カメラを開始」を押します。
4. 背景写真を複数選択すると、人物の背景でスライドショーが再生されます。

## GitHub Pagesで公開する場合

1. GitHubで新しいリポジトリを作成します。
2. `index.html` をアップロードします。
3. Settings → Pages を開きます。
4. Branchを `main`、folderを `/root` にして保存します。
5. 数分後に `https://ユーザー名.github.io/リポジトリ名/` で開けます。

## 注意

- スマホのカメラは、基本的にHTTPSまたはlocalhostでないと使えません。
- iPhoneではSafari推奨です。
- ChatGPTアプリ内プレビューやファイル直開きでは、カメラが起動しないことがあります。
